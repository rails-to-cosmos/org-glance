#+TITLE: Changelog

All notable, user-facing changes to org-glance are recorded here, in the spirit
of [[https://keepachangelog.com/][Keep a Changelog]].  Versions are date-based
=MAJOR.MINOR.PATCH.BUILD.YYYYMMDD.REV= (see the =Makefile= and the package
header).  Design detail lives in [[file:docs/proposals/][docs/proposals/]]; this file is the summary.

* Unreleased

** Added
- Encrypt / decrypt a headline straight from the table view (=org-glance-table=):
  =#= toggles encryption of the row at point, prompting for a password
  (confirmed when encrypting); =C-u #= re-keys an encrypted headline (old
  password, then new).
- =Enc= column in the table, showing 🔒 for encrypted headlines (sortable, so
  encrypted rows group together).
- Store-level crypto API: =org-glance-material:crypt-set= (encrypt/decrypt) and
  =org-glance-material:crypt-rekey= (change password), sharing one guarded
  transform helper.

** Changed
- The materialized-buffer password lock moved from =C-c C-l= (org's
  =org-insert-link=) to =C-c #=.

** Fixed
- Capturing from the table no longer raises a "Buffer modified; kill anyway?"
  prompt (the temporary capture buffer is discarded cleanly).

* 0.6.0.0.20260716.0

** Added
- All-tags overview (=org-glance-tags=, bound =t=): a table of tags with per-tag
  metrics — headline count, todo-state breakdown, configured cycle, and
  event-tracked created/modified timestamps.  Row actions: open the tag's table
  (RET) or overview (=o=), add (=+=) / remove (=-=) a tag.
- Git-sync conflict auto-resolution.  The =meta/*.jsonl= WAL heals via a
  =merge=union= git driver plus a marker-stripping recovery on open; the
  single-form =config/*.eld= sidecars (e.g. =tag-metrics.eld=) heal on read
  through a reusable, format-agnostic conflict toolkit in =org-glance-utils=
  (=org-glance-conflict-resolution= governs both).  See
  [[file:docs/proposals/2026-07-15-eld-sidecar-conflict-resolution.done.org][eld-sidecar-conflict-resolution]].
- =agnostic-llm= integration (=org-glance-llm=, the =l= action): opens an LLM
  session in a headline's data directory; the =*llm:…*= buffer is named for the
  headline's title slug, not the data directory's content hash.
- =table-view= and =agnostic-llm= consumed as external ELPA/git dependencies
  rather than vendored.

** Changed
- Date-based versioning (=MAJOR.MINOR.PATCH.BUILD.YYYYMMDD.REV=), stamped by the
  =Makefile= bumpers (=make major|minor|patch|build|rev=).
- Main entry point renamed =org-glance-form-action= → =org-glance-transient=
  (autoloaded, with all its suffix commands).
- Session primitives moved to =org-glance-core= so autoloaded commands
  initialise cleanly on a fresh install.
